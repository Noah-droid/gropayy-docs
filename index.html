Gropay API Documentation

Introduction

Welcome to the Gropay API, your gateway to seamless financial services integration. Our API provides a secure, scalable, and developer-friendly way to manage user accounts, payments, and transactions within the Gropay ecosystem.

This documentation will guide you through authentication, user registration, transactions, and more. Whether you're building a fintech app, integrating payment processing, or managing user identities, our API is designed to help you move fast with confidence.

Key Features

  • User Management – Register, authenticate, and manage users securely.
  • Payments & Transactions – Process payments, manage transactions, and track balances.
  • Security & Compliance – Built with robust security measures, including BVN verification.
  • Webhooks & Notifications – Get real-time updates on transactions and user actions.

Getting Started

To begin using the Gropay API: 1. Sign up for API access and get your credentials. 2. Authenticate using JWTs or OAuth. 3. Start integrating using the endpoints documented below.
